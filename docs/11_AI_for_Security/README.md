# 11 AI for Security：AI 辅助安全研究

本章用于学习如何把 AI 作为工控安全研究助手：整理资料、辅助代码审计、解释协议字段、生成测试计划、润色报告、构建研究 Agent 工作流。

核心原则：AI 可以提速，但安全结论必须由证据验证。

## 本章能力目标

- 能设计适合漏洞研究、协议分析、逆向复盘的 Prompt。
- 能用 AI 辅助阅读公告、源码、PoC、抓包摘要和报告草稿。
- 能知道 AI 的边界：幻觉、版本过期、上下文缺失、安全误判。
- 能建立 “AI 建议 -> 人工验证 -> 证据沉淀” 的工作流。

## 推荐使用方式

- 用 AI 做资料归纳，不让 AI 代替证据。
- 用 AI 生成检查清单，再人工执行。
- 用 AI 解释代码片段，再用调试和测试验证。
- 用 AI 生成报告初稿，再补充真实截图、日志、调用链和版本。
