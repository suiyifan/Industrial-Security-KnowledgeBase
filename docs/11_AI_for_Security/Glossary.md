# 术语表

| 术语 | 解释 |
| --- | --- |
| Prompt | 给 AI 的任务指令和上下文 |
| Hallucination | AI 编造事实或错误结论 |
| Human-in-the-loop | 人在流程中验证和决策 |
| RAG | 检索增强生成，把外部资料检索后提供给 AI |
| Agent | 能按步骤调用工具完成任务的 AI 工作流 |
| 事实 | 有证据支撑的内容 |
| 推测 | 基于现有材料的假设 |
| 待验证 | 需要进一步实验或资料确认的问题 |
| 置信度 | 对某个判断可靠程度的描述 |
| 代码审计 | 检查代码中安全缺陷的过程 |
