# 工控 Web 管理界面风险

越来越多工业设备提供 Web 管理界面，例如工业网关、交换机、防火墙、PLC、远程运维设备、摄像头和安全设备。

常见功能：

- 登录管理
- 网络配置
- 协议转换配置
- 用户管理
- 固件升级
- 日志下载
- 远程诊断
- VPN 配置

管理界面被攻陷后，攻击者可能修改网关协议转换、获取 PLC 地址、开启远程访问、修改防火墙策略或上传恶意固件。

> 面试表达：工控 Web 管理界面通常连接真实工业资产。评估时我会看默认密码、认证、固件上传、配置备份、远程诊断和管理面暴露范围。
