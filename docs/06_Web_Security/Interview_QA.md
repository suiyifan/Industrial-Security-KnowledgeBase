# 06 章面试问答

## 1. JD 为什么要求 Web 漏洞？

> 因为工控平台、工业网关、安全设备和远程运维系统都有 Web 管理界面。Web 漏洞可能成为进入工业网络的入口。

## 2. SQL 注入在工控场景有什么特殊影响？

> 可能泄露资产清单、点表、报警、账号和远程运维配置，帮助攻击者理解控制系统。

## 3. 文件上传为什么危险？

> 不只是 WebShell，如果文件交换或工程备份入口有问题，恶意文件可能进入工程站或影响固件、配置包。

## 4. RCE 如何映射到工控影响？

> 先看资产角色。RCE 在工业网关上可能接触 PLC 地址和协议转换；在 SCADA Server 上可能影响采集、报警和趋势。

## 5. SSRF 为什么要关注服务器位置？

> 如果服务器在工业 DMZ 或生产网，SSRF 可能绕过边界访问内部 HMI、网关或控制资产。
