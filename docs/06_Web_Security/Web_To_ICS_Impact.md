# Web 漏洞到工控影响的映射

## 映射方法

```text
Web 漏洞 -> 受影响资产 -> 获得权限 -> 可访问数据或功能 -> 工控资产路径 -> 生产影响
```

| Web 漏洞 | 工控影响 |
|---|---|
| SQL 注入 | 泄露资产、点表、报警、账号 |
| 文件上传 | WebShell、恶意文件进入工程环境 |
| 越权 | 普通用户执行工程师或管理员操作 |
| RCE | 接管网关、SCADA、DMZ 服务器 |
| SSRF | 从 DMZ 探测内网控制资产 |
| XSS | 劫持操作员会话 |

报告模板：

```text
该 Web 漏洞位于【资产】。
该资产在工控架构中承担【功能】。
攻击者利用后可【技术动作】。
进一步可能访问【工程文件/点表/控制网/远程运维】。
最终可能导致【工控影响】。
```
