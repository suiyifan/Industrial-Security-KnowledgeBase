# 06 章术语表

## SQL Injection

SQL 注入，攻击者改变数据库查询逻辑。

## WebShell

上传到服务器上的恶意脚本后门。

## IDOR

不安全直接对象引用，常见越权类型。

## RCE

远程命令或代码执行。

## SSRF

服务端请求伪造，让服务器访问攻击者指定地址。

## XSS

跨站脚本，可用于劫持用户会话或执行前端恶意逻辑。
