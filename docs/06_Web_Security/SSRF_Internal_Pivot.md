# SSRF、内网探测与工控边界

SSRF 是服务端请求伪造。攻击者让服务器代替自己访问内部地址。

常见入口包括 URL 预览、远程文件导入、Webhook、图片抓取、在线诊断、接口代理。

如果 Web 服务位于工业 DMZ 或生产管理网，SSRF 可能用于：

- 探测控制网资产。
- 访问内部管理界面。
- 读取云元数据或凭据。
- 绕过边界访问工业网关。

> 面试表达：SSRF 在工控环境中要重点看服务器所在网络位置。如果它在 DMZ 或生产网，攻击者可能利用它作为内网探测点，访问外部无法直接访问的工业网关、HMI 或管理接口。
