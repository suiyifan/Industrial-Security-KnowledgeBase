# 越权、认证绕过与权限模型

越权漏洞是用户可以访问或操作本不属于自己的资源。认证绕过是攻击者无需有效身份即可进入受保护功能。

工控平台常见角色：

- 查看人员
- 操作员
- 工程师
- 管理员
- 厂商维护人员

如果后端只靠前端隐藏按钮，攻击者可直接调用 API。

工控风险：

- 普通用户修改设备参数。
- 查看或下载工程文件。
- 打开远程维护通道。
- 修改网关、防火墙或采集配置。

> 面试表达：工控系统里的越权风险要看越权功能是什么。如果越权修改设定值、远程运维配置或报警屏蔽，可能直接影响控制和安全监测。
