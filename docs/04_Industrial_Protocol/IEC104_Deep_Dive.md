# IEC 60870-5-104：电力远动协议

## 1. 原理解释

IEC 60870-5-104 常用于电力远动通信，运行在 TCP/IP 上，常见端口为 2404。

它服务于调度中心、变电站、RTU、远动装置之间的数据采集和控制。

## 2. 工程实现

典型链路：

```text
调度主站
  -> TCP 2404
  -> IEC 104
  -> 子站 / RTU / 远动装置
  -> 保护测控 / 开关 / 遥测遥信
```

电力通信中常见概念：

- 遥测：模拟量测量，如电压、电流、功率。
- 遥信：开关状态，如断路器位置。
- 遥控：远程控制，如分合闸。
- 遥调：远程调节。

## 3. 数据流

```text
子站上送遥测/遥信
  -> 主站显示电网状态
主站下发遥控
  -> 子站执行选择/执行流程
  -> 现场开关动作
```

遥控通常比遥测风险高，因为它会影响现场设备状态。

## 4. 安全风险

- 未授权主站连接。
- 遥测数据伪造，影响调度判断。
- 遥信状态伪造，隐藏开关真实状态。
- 遥控命令滥用，导致设备误分合。
- 明文传输，缺少强认证。
- 站点边界暴露。

## 5. 抓包观察点

过滤：

```text
tcp.port == 2404
iec104
```

重点关注：

- 连接建立。
- 总召唤。
- 遥测上送。
- 遥信变位。
- 遥控选择/执行。
- 异常高频或非主站来源。

## 6. 工业案例

如果未知主机向子站发起 IEC 104 连接，并出现遥控相关 APDU，应按高风险处理：

```text
未知主机疑似模拟主站与远动装置通信，可能具备下发遥控命令的能力。若命令被执行，可能影响开关状态和调度安全。
```

## 7. 面试表达

### 60 分答案

> IEC 104 是电力远动协议，常用 TCP 2404，主要承载遥测、遥信、遥控、遥调等数据。

### 90 分答案

> 分析 IEC 104 时，我会区分上送数据和下发控制。遥测、遥信影响调度感知，遥控会影响现场开关设备状态。安全风险不只是端口暴露，而是未知主站连接、遥控命令滥用、遥信遥测伪造对调度判断和现场设备造成影响。

## 8. 实操建议

- 学习 IEC 104 APDU 类型。
- 抓取公开样例 PCAP。
- 区分遥测、遥信、遥控。
- 写一张“电力数据类型 -> 安全影响”表。
