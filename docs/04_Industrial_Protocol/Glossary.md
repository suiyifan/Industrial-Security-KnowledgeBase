# 04 章术语表

## Modbus

寄存器模型工业协议，常见于 PLC、仪表、网关和远程 I/O。

## S7Comm

西门子 S7 PLC 常见通信协议，常见端口 TCP 102。

## OPC UA

工业互联协议，使用节点模型，支持较完整安全机制。

## IEC 104

电力远动协议，常见端口 TCP 2404。

## Profinet

工业以太网协议，常见于实时 I/O 和西门子生态。

## EtherNet/IP

基于 CIP 的工业以太网协议，常见于罗克韦尔生态。

## EtherCAT

高实时工业以太网，常用于运动控制。

## Function Code

功能码，表示协议执行的操作类型。

## Register

寄存器，Modbus 中用于表示数据变量的基本单位。

## DB

Data Block，西门子 PLC 中保存业务状态和实例数据的数据块。

## Node

OPC UA 地址空间中的节点。

## Browse

浏览 OPC UA 地址空间。

## Read / Write

读写操作。写操作通常比读操作风险更高。
