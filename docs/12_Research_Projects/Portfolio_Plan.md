# 作品集计划

## Project 1：Modbus TCP 协议解析与探测工具

目标：

- 展示 Python、Socket、协议理解、工具开发能力。

功能：

- 构造读取保持寄存器请求
- 解析响应
- 输出功能码、寄存器地址、返回值

## Project 2：工控 CVE 分析报告

目标：

- 展示漏洞分析、复现、报告能力。

内容：

- 选择公开工控 CVE
- 整理影响版本
- 复现或做静态分析
- 写工控场景影响

## Project 3：模拟工控安全评估报告

目标：

- 展示安全评估、审计、体系化表达能力。

内容：

- 资产清单
- 网络拓扑
- 风险点
- 改进建议
- 与 IEC 62443 的对应关系
