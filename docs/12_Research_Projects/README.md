# 12 Research Projects：研究项目与作品集

本章把前面 01-11 章的知识沉淀成 GitHub 作品集。目标不是堆文件，而是让面试官看到：你能围绕一个工控安全问题完成环境搭建、分析、验证、报告和复盘。

## 本章能力目标

- 规划适合求职展示的工控安全项目。
- 把理论、工具、协议、漏洞和报告连接起来。
- 形成可讲述的项目经历。
- 让每个项目都有 README、证据、结论和限制。

## 作品集原则

- 少而完整。
- 只展示授权和仿真环境。
- 证据清晰。
- 风险边界明确。
- 能用 2 分钟讲清楚背景、方法、结果和反思。
