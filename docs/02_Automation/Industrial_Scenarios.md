# 工业行业场景

## 1. 为什么要理解行业场景

HR 提到公司客户包括电力、石油、石化、冶金、水处理等方向。不同场景的控制对象、系统架构和安全后果不一样。

面试时如果能把漏洞影响和行业场景联系起来，会比泛泛说“有风险”更有说服力。

## 2. 电力

常见对象：

- 发电机
- 变电站
- 保护装置
- RTU
- 调度系统
- 电力监控系统

常见协议：

- IEC 60870-5-104
- IEC 61850
- Modbus
- DNP3

安全影响：

- 遥测数据异常。
- 遥控命令被滥用。
- 保护和监控信息失真。
- 调度判断受到影响。

## 3. 石油石化

常见对象：

- 反应釜
- 加热炉
- 压缩机
- 泵
- 阀门
- 储罐
- 管线

常见系统：

- DCS
- SIS
- SCADA
- 火气系统
- 历史数据库

安全影响：

- 温度、压力、流量异常。
- 生产中断。
- 设备损坏。
- 安全联锁触发或失效。
- 高风险场景下可能涉及人身和环境安全。

## 4. 冶金

常见对象：

- 高炉
- 轧机
- 加热炉
- 连铸设备
- 风机
- 电机

常见系统：

- PLC
- DCS
- 传动控制系统
- HMI

安全影响：

- 设备节奏异常。
- 温度控制异常。
- 传动系统异常。
- 产线停机。

## 5. 水处理

常见对象：

- 水泵
- 阀门
- 液位计
- 流量计
- 加药系统
- 曝气系统

常见系统：

- PLC
- SCADA
- HMI
- 远程站点

安全影响：

- 液位异常。
- 加药比例异常。
- 泵站异常启停。
- 水质或供水稳定性受影响。

## 6. 智能制造 / 离散制造

常见对象：

- 机器人
- 产线设备
- 传送带
- CNC
- 视觉检测
- 伺服系统

常见系统：

- PLC
- MES
- HMI
- 工业网关

安全影响：

- 产线停机。
- 产品质量异常。
- 机器人动作异常。
- 生产数据被篡改。

## 7. 行业场景对安全分析的影响

同一个漏洞，在不同行业里的影响不一样。

例子：未授权写变量。

| 场景 | 可能影响 |
|---|---|
| 水处理 | 泵启停、阀门开度、加药量 |
| 石化 | 温度压力设定、联锁条件、阀门状态 |
| 电力 | 遥控、遥测、保护信息 |
| 制造 | 产线节拍、设备动作、质量检测 |

## 8. 面试表达

可以这样说：

> 我会结合行业场景分析工控漏洞影响。例如同样是未授权写变量，在水处理场景可能影响泵和阀门，在石化场景可能影响温度压力设定和安全联锁，在电力场景则可能影响遥控遥测和调度判断。工控安全报告需要把技术影响翻译成业务和生产影响。
