# 工业自动化基础

## 必须理解的对象

- PLC：可编程逻辑控制器，负责现场控制逻辑。
- DCS：分布式控制系统，常见于流程工业。
- SCADA：监控与数据采集系统，负责远程监控、告警和数据采集。
- HMI：人机界面，操作员查看状态和下发指令。
- 工程站：配置、编程、下载控制逻辑的工作站。
- 操作员站：用于日常监控和操作。
- 历史数据库：存储过程数据、告警、趋势。

## 面试表达

工控安全不是只看单台主机，而是要看控制链路：工程站如何配置 PLC，HMI 如何显示状态，SCADA 如何采集数据，协议如何在网络上传输，漏洞最终会不会影响生产过程、安全联锁或设备状态。
