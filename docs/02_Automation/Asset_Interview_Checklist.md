# 资产访谈与现场调研问题清单

这份清单用于让你在面试中表现出“知道项目怎么问、怎么查、怎么落地”。

## 1. 总体架构问题

- 当前系统服务哪条产线或哪个工艺单元？
- 是否有最新网络拓扑图？
- 控制网络和办公网络如何隔离？
- 是否有工业 DMZ？
- 是否存在跨区域通信？
- 哪些系统是关键生产系统？
- 哪些系统停机会造成生产影响？

## 2. PLC / 控制器问题

- PLC 型号和厂商是什么？
- PLC 是否有以太网模块？
- PLC 是否开启 Web、FTP、诊断、工程服务？
- PLC 程序是否有上传 / 下载保护？
- 工程下载是否需要审批？
- 是否允许在线修改？
- 是否使用 Force？
- 是否记录 Force 和下载操作？
- PLC 程序是否有基线备份？

## 3. HMI / SCADA 问题

- HMI 是否能下发控制命令？
- HMI 是否有用户分级？
- 是否存在共用账号？
- 关键操作是否二次确认？
- 报警是否可屏蔽？
- 报警屏蔽是否审批？
- 组态工程文件存放在哪里？
- HMI 与 PLC 使用什么协议？

## 4. DCS 问题

- DCS 厂商和版本是什么？
- 工程师站有几台？
- 谁有工程师权限？
- 控制策略变更流程是什么？
- 历史站保存多久数据？
- 报警是否分级？
- 是否存在长期旁路或抑制？
- 操作员站是否允许外接 U 盘？

## 5. 远程运维问题

- 是否有厂商远程运维？
- 远程入口是 VPN、堡垒机、RDP 还是专线？
- 是否启用多因素认证？
- 远程账号是否定期回收？
- 远程操作是否录像或审计？
- 紧急远程访问是否有审批流程？

## 6. 账号权限问题

- 是否存在默认账号？
- 是否存在共享账号？
- 密码是否定期更换？
- 离职人员账号是否删除？
- 操作员和工程师权限是否分离？
- 管理员账号是否日常使用？

## 7. 备份恢复问题

- PLC / DCS / SCADA 工程是否定期备份？
- 备份是否离线保存？
- 是否做过恢复演练？
- 备份版本和现场运行版本是否一致？
- 历史数据库是否备份？
- 勒索场景下如何恢复？

## 8. 安全监测问题

- 是否监测工业协议流量？
- 是否有工控 IDS？
- 是否有异常报警联动？
- 是否记录关键操作日志？
- 是否能发现工程下载、变量写入、PLC 状态切换？

## 9. 面试用法

如果面试官问“你会怎么做工控安全评估”，可以回答：

> 我会先做访谈和资料收集，问题会覆盖架构、PLC、HMI/SCADA、DCS、远程运维、账号权限、备份恢复和安全监测。比如对 PLC，我会问是否开启工程服务、是否有下载保护、是否允许 Force、是否有程序基线；对 HMI，我会问是否有共用账号、关键操作是否二次确认、报警是否可屏蔽。这样能把技术检查和生产风险对应起来。
