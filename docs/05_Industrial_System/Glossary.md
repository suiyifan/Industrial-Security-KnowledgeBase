# 05 章术语表

## 工程站

用于编程、配置、调试、上传下载工程的工作站。

## 操作员站

用于日常监控、报警处理和设备操作的工作站。

## Historian

历史数据库，用于保存过程数据、报警、趋势和事件。

## RTU

远程终端单元，常用于远程站点数据采集和控制。

## IED

智能电子设备，常见于电力保护、测控场景。

## 工业网关

连接不同协议、网络或系统的边界设备。

## 基线核查

检查系统配置是否满足最低安全要求。

## 固件

运行在设备内部的底层软件。
