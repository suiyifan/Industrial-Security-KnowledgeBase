# 工业网关与安全网关

## 1. 原理解释

工业网关连接不同协议、不同网络和不同安全域。安全网关用于隔离、访问控制、协议审计或数据交换。

## 2. 工程实现

工业网关常用于 Modbus RTU/TCP 转换、OPC UA 发布、MQTT/HTTP 上传、串口转以太网、边缘计算和远程维护。

安全网关常用于白名单访问控制、工业协议审计、单向传输、文件交换和边界隔离。

## 3. 安全风险

- 默认密码。
- Web 管理漏洞。
- 配置泄露点表和 PLC 地址。
- 协议转换扩大攻击面。
- 云连接引入外部风险。
- 写操作未限制。

## 4. 面试表达

> 工业网关是边界资产，既可能连接 PLC，也可能连接上层平台或云端。评估时我会确认它是只读采集还是可写控制，管理界面是否暴露，配置是否泄露点表和地址，以及协议转换是否扩大了低安全协议的暴露范围。
