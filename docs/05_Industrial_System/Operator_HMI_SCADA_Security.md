# 操作员站、HMI 与 SCADA Server

## 1. 原理解释

操作员站和 HMI 是人和控制系统交互的入口。SCADA Server 负责集中采集、报警、趋势和调度。

## 2. 工程实现

HMI/SCADA 常见功能包括工艺画面、实时变量显示、设备启停、参数输入、报警确认、趋势查看、用户权限、脚本和报表。

## 3. 数据流

```text
操作员点击按钮
  -> HMI 写变量或执行脚本
  -> SCADA 通信驱动
  -> PLC / RTU / DCS
  -> 设备动作
```

## 4. 安全风险

- 共用账号。
- HMI 越权操作。
- 按钮背后脚本可执行危险动作。
- 报警屏蔽。
- 趋势数据篡改。
- Web 管理界面漏洞。
- 远程桌面暴露。

## 5. 面试表达

> HMI 不只是显示界面，它经常能写变量、执行脚本和下发控制命令。评估时我会追踪画面按钮背后的变量、脚本、权限和日志，判断它是否可能影响设备启停、模式切换、设定值或报警确认。
