# RTU、IED、远程站点与边缘设备

## 1. 原理解释

RTU 和 IED 常用于远程站点、电力、水务、油气管线等分散场景。它们负责采集现场数据并接受主站调度。

## 2. 工程实现

常见环境包括泵站、阀室、变电站、光伏/风电场站、环保监测站和管线远程站点。

通信方式可能是专线、VPN、4G/5G、IEC 104、Modbus、DNP3 或厂商私有协议。

## 3. 安全风险

- 物理环境无人值守。
- 蜂窝路由器暴露。
- 默认密码。
- 远程维护常开。
- 站点通信缺少强认证。
- 遥控命令风险。

## 4. 面试表达

> 远程站点的特点是分散、无人值守、依赖远程通信。评估时我会重点看 4G/5G 路由器、VPN、默认密码、遥控权限和本地保护逻辑，避免外部入口直接影响现场设备。
