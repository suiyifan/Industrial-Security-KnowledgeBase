# 05 章面试问答

## 1. 为什么工程站是高价值资产？

> 工程站保存工程文件、点表、控制逻辑和通信配置，还能上传下载工程、在线监控和 Force 变量。攻击工程站可能让攻击者理解并修改控制系统。

## 2. HMI 的安全风险是什么？

> HMI 不只是显示界面，通常能写变量、执行脚本、确认报警、修改设定值和切换模式。风险要看按钮背后的动作和权限。

## 3. 历史数据库为什么重要？

> 历史库保存过程趋势、报警和操作记录，是事故追溯证据源。如果被篡改，可能隐藏攻击影响；如果双归属网络，还可能成为跳板。

## 4. 工控补丁为什么难？

> 因为生产连续性、兼容性、厂商认证和停机窗口限制。不能直接自动更新，需要测试、审批、备份、回退和复测。

## 5. 基线核查怎么体现工控特色？

> 要结合资产角色解释影响。工程站的 RDP 暴露可能导致工程下载风险；HMI 共用账号可能影响设备操作；历史库备份缺失会影响事故恢复。
