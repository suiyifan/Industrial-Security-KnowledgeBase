# 01 章术语表

## Git

版本控制工具，用来记录文件变化。

## Repository

仓库，被 Git 管理的项目目录。

## Commit

一次正式提交，表示项目在某个时间点的状态。

## Branch

分支，用来并行开发或尝试不同方向。

## Linux

常见操作系统，广泛用于服务器、安全工具、嵌入式设备。

## Shell

命令行解释器，用来执行命令。

## Process

进程，正在运行的程序。

## Port

端口，用于区分一台机器上的不同服务。

## IP Address

IP 地址，网络中设备的地址。

## TCP

面向连接的传输层协议，可靠、有顺序。

## UDP

无连接传输层协议，速度快但不保证可靠。

## DNS

域名系统，把域名解析成 IP 地址。

## HTTP

Web 通信协议，用于浏览器和服务器之间传输数据。

## ARP

地址解析协议，在局域网中通过 IP 找到 MAC 地址。

## ICMP

网络控制消息协议，`ping` 常用它测试连通性。

## Socket

程序进行网络通信的接口。

## Packet

数据包，网络中传输的数据单位。

## PCAP

抓包文件格式，用于保存网络流量。

## Wireshark

抓包与协议分析工具。

## Display Filter

Wireshark 显示过滤器，用于从已抓取的数据包中筛选内容。

## Capture Filter

Wireshark 抓包过滤器，用于抓包前决定只捕获哪些流量。

## Modbus TCP

常见工业协议，默认使用 TCP 502 端口。

## S7comm

西门子 PLC 通信常见协议，常见于 TCP 102 端口。

## OPC UA

工业互联通信协议，常见默认端口为 TCP 4840。

## IEC 104

电力行业常见远动通信协议，常见端口为 TCP 2404。
