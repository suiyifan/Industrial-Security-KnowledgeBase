# 01 章学习地图：计算机与安全基础

这一章的目标不是“学完计算机全部基础”，而是为后面的工控安全研究打底。你要能做到：

- 会用 Python 写小工具。
- 会用 Git 管理学习记录和作品集。
- 会在 Linux 里查进程、看端口、找日志、改权限。
- 看得懂 TCP/IP、HTTP、DNS、ARP、ICMP 的基本过程。
- 会用 Wireshark 抓包、过滤、解释常见网络流量。

## 学习顺序

1. 先学 Git：保证每天的学习都有版本记录。
2. 再学 Linux：后续安全工具、靶场、服务器都离不开它。
3. 再学 Python：用来写端口扫描、协议解析、日志处理等工具。
4. 再学网络基础：理解数据如何从一台机器到另一台机器。
5. 最后学 Wireshark：把抽象网络知识变成可见的数据包。

## 每个知识点怎么学

每个小节都按四层学习：

1. 它是什么：先用白话理解。
2. 为什么岗位需要：连接到工控安全场景。
3. 必须掌握的细节：列出面试和实践会用到的点。
4. 动手练习：用命令、脚本或抓包验证。

## 本章完成标准

完成本章后，你应该能独立完成这些任务：

- 创建 Git 仓库并提交学习笔记。
- 在 Linux 中查看 IP、端口、进程、文件权限。
- 用 Python 写 TCP 客户端 / 服务端。
- 解释 TCP 三次握手和四次挥手。
- 用 Wireshark 抓 DNS、HTTP、TCP、ICMP 包。
- 说清楚端口扫描器的基本原理。

## 和目标岗位的关系

JD 中要求“精通至少一种编程语言”“具备算法设计能力”“有良好编程习惯”“熟悉常见 Web / 系统漏洞”“工具开发”。这些能力都依赖本章基础。

HR 提到岗位偏工控系统安全研究、漏洞挖掘、协议分析。工控协议本质上也是网络通信协议，后续学 Modbus TCP、S7、OPC UA、IEC 104 前，必须先理解 TCP/IP、端口、报文、抓包和 Socket。
