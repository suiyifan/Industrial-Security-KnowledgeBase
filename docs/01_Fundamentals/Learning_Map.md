# 01 篇学习地图：计算机科学与安全工程基础

本篇不是“开始阶段所以简单”，而是整套工控安全知识体系的承重层。后续看到的 PLC 内存、工业协议字段、固件漏洞、Web 管理面、证书认证、历史数据库和事件响应，都可以追溯到本篇的某个底层模型。

## 1. 能力结构

```text
数据表示与计算机组成
        |
        v
操作系统原理 ----> C 与内存安全 ----> 逆向 / 漏洞研究
        |
        +----> Linux / Windows ----> 主机安全 / 固件 / 应急
        |
Python / Git ----------------------> 工具开发 / 研究工程化
        |
网络通信 -> 网络服务 -> Wireshark -> 工业协议 / 网络监测
        |
数据库 / 密码学 / 身份 / 安全工程 -> Web / 平台 / 安全评估
        |
虚拟化与日志取证 -----------------> 可复现实验 / 证据闭环
```

## 2. 四个学习阶段

### 阶段 A：程序与系统为什么这样运行

1. 计算机组成与数据表示；
2. 操作系统原理与运行时模型；
3. Python 安全研究基础；
4. C 语言、程序内存与内存安全。

完成后应能解释整数、字符串和协议字段如何进入内存，程序如何通过系统调用访问文件和网络，进程崩溃为什么可能与越界访问有关。

### 阶段 B：真实主机如何管理和排查

5. Linux 系统管理与安全分析；
6. Windows 系统管理与安全分析；
7. Git、GitHub 与研究证据管理。

完成后应能围绕用户、权限、进程、服务、网络、日志和持久化位置进行主机排查，并用版本控制保存代码、配置与分析证据。

### 阶段 C：系统如何通信与建立信任

8. TCP/IP 与网络通信基础；
9. 网络服务、Web 与远程管理基础；
10. 数据库与工业数据基础；
11. 密码学、PKI 与身份基础；
12. 网络安全工程基础。

完成后应能解释一次跨网段、带认证和加密的应用访问，区分网络可达、服务可用、身份通过、对象授权和业务执行五种不同状态。

### 阶段 D：如何安全实验并形成证据

13. 虚拟化、容器与隔离实验环境；
14. Wireshark 与工业流量分析；
15. 日志、可观测性与数字取证基础。

完成后应能构建隔离靶场，采集 PCAP、系统日志和应用日志，校准时间并形成可复核的事件时间线。

## 3. 每个知识点的学习深度

每个关键概念至少达到五层：

1. **概念层**：能准确说明它是什么、解决什么问题；
2. **原理层**：能解释为什么如此设计、有哪些约束；
3. **观测层**：知道用什么命令、日志或抓包看到它；
4. **诊断层**：能根据输出区分正常、故障和安全异常；
5. **迁移层**：能把结论映射到工程站、SCADA、网关、PLC 通信和物理过程。

例如学习 `ss -lntup`，不应停留在“查看端口”，而要理解监听 Socket 属于哪个网络命名空间、对应哪个进程和权限、是否暴露在预期接口、是否符合通信矩阵，以及异常服务对控制区意味着什么。

## 4. 本篇完成标准

### 计算机与程序

- 手工完成二进制、十六进制、补码、字节序和位运算转换；
- 解释 CPU、缓存、内存、MMU、中断、DMA 和 I/O 的基本关系；
- 解释进程、线程、虚拟内存、系统调用、文件描述符和并发；
- 使用 Python 可靠处理二进制协议、日志、配置和命令行参数；
- 阅读基础 C 代码并识别越界、生命周期、整数和格式化风险。

### 系统与工程

- 在 Linux 和 Windows 中调查账户、权限、进程、服务、连接和日志；
- 区分普通配置错误、资源故障、异常持久化与权限滥用；
- 使用 Git 管理章节、代码、配置、实验数据和修复差异；
- 构建带快照、网络隔离和恢复步骤的实验环境。

### 网络与安全

- 解释 Ethernet、VLAN、ARP、路由、TCP/UDP、DNS、HTTP/TLS；
- 解释 SQL 事务、数据库权限、备份恢复和工业历史数据链；
- 解释哈希、加密、签名、证书、认证与授权的边界；
- 建立资产、威胁、漏洞、利用条件、安全控制和后果模型；
- 使用 Wireshark 和日志还原一个跨层事件，并注明证据局限。

## 6. 推荐学习方法

每章都保留四类产出：一张原理图、一组可复现实验、一份命令或代码结果、一段面试表达。命令输出必须解释字段含义和下一步判断；实验必须记录环境、输入、预期、实际、证据和清理步骤。

不要把命令数量当成掌握程度。真正的完成标准是：面对一个未知异常，能够选择正确观测点，提出可证伪假设，采集最小充分证据，并说明它对工业系统和物理过程的实际意义。
