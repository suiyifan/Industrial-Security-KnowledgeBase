# 工业交换机、路由与 VLAN

## 1. 工业交换机是什么

工业交换机用于连接工业网络中的设备。相比普通交换机，它通常更适应工业环境：

- 宽温
- 抗干扰
- 冗余环网
- DIN 导轨安装
- 工业协议特性
- 更高稳定性

安全分析中，交换机决定了很多设备是否在同一个二层网络里。

## 2. VLAN

VLAN 用于在交换机上划分逻辑网络。

示例：

| VLAN | 用途 |
|---|---|
| VLAN 10 | 监控层 |
| VLAN 20 | 控制层 |
| VLAN 30 | 工程维护 |
| VLAN 40 | 安全监测 |

注意：VLAN 是隔离手段之一，但不是完整安全边界。如果三层互通没有访问控制，VLAN 之间仍然可以互访。

## 3. Trunk 与 Access

Access 端口：

- 接普通终端或设备。
- 通常只属于一个 VLAN。

Trunk 端口：

- 用于交换机之间连接。
- 可承载多个 VLAN。

安全关注：

- Trunk 是否不必要地延伸到普通设备端口。
- 未使用端口是否关闭。
- 默认 VLAN 是否被滥用。

## 4. 路由

路由决定不同网段之间如何通信。

评估时关注：

- 哪些网段可以互通？
- 路由是否绕过防火墙？
- 是否存在静态路由直达控制层？
- 是否存在临时路由长期保留？

## 5. 工业环网

工业网络常用冗余环网提高可用性。

目的：

- 单条链路断开时通信不完全中断。
- 提高生产网络可靠性。

安全关注：

- 环网协议是否稳定。
- 错误接线是否导致风暴。
- 是否有广播风暴保护。
- 是否有端口环路保护。

## 6. 常见交换机安全配置

- 禁用未使用端口。
- 管理口限制访问源。
- 修改默认密码。
- 开启日志。
- 开启 SNMPv3，避免 SNMPv1/v2 弱社区字符串。
- 管理 VLAN 单独隔离。
- 配置端口描述。
- 备份配置。

## 7. 常见风险

- 交换机默认密码。
- Web 管理界面暴露。
- Telnet 明文管理。
- SNMP community 为 `public`。
- 未使用端口可随意接入。
- VLAN 规划混乱。
- 控制网和办公网三层互通无 ACL。

## 8. 面试表达

> 工业交换机不仅是连线设备，也影响安全域和通信路径。评估时我会看 VLAN 划分、Trunk 范围、未使用端口、管理口访问控制、默认密码、SNMP 配置和路由路径。尤其要确认 VLAN 之间是否通过 ACL 或防火墙做了访问控制，避免逻辑分区变成形式化分区。
