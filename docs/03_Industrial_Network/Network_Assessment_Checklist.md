# 网络评估检查清单

## 1. 拓扑与资产

- [ ] 是否有最新网络拓扑图。
- [ ] 是否标注安全域。
- [ ] 是否标注边界设备。
- [ ] 是否标注关键资产。
- [ ] 是否标注远程运维入口。
- [ ] 是否存在未知资产。

## 2. 分区分域

- [ ] 办公网与控制网是否隔离。
- [ ] 是否有工业 DMZ。
- [ ] DMZ 是否独立区域。
- [ ] 控制层和监控层是否隔离。
- [ ] SIS / ESD 是否单独隔离。
- [ ] 是否存在双网卡跨接。

## 3. 通信矩阵

- [ ] 是否有通信矩阵。
- [ ] 是否记录源、目的、协议、端口、方向。
- [ ] 是否标注读写权限。
- [ ] 是否确认业务必要性。
- [ ] 是否存在跨域异常通信。
- [ ] 是否存在办公网访问 PLC。

## 4. 边界防护

- [ ] 防火墙策略是否最小化。
- [ ] 是否存在 `any any allow`。
- [ ] 临时策略是否长期保留。
- [ ] 是否记录日志。
- [ ] 是否限制工程协议。
- [ ] 是否监测 PLC 写操作。

## 5. 远程运维

- [ ] 是否有远程运维清单。
- [ ] 是否按需开通。
- [ ] 是否启用 MFA。
- [ ] 是否经过堡垒机。
- [ ] 是否有会话审计。
- [ ] 是否允许文件传输。
- [ ] 账号是否定期回收。

## 6. 交换机与路由

- [ ] VLAN 是否合理。
- [ ] VLAN 间是否有 ACL。
- [ ] 未使用端口是否关闭。
- [ ] 管理口是否限制访问。
- [ ] 默认密码是否修改。
- [ ] SNMP 是否安全配置。

## 7. 工业网关

- [ ] 网关是否跨安全域。
- [ ] 是否允许写操作。
- [ ] 管理界面是否暴露。
- [ ] 默认密码是否修改。
- [ ] 是否连接云平台。
- [ ] 配置是否加密备份。

## 8. 监测与审计

- [ ] 是否部署工业 IDS。
- [ ] 是否有镜像口或 TAP。
- [ ] 是否监测跨区通信。
- [ ] 是否监测工程下载。
- [ ] 是否监测 PLC 状态变化。
- [ ] 是否有日志留存周期。

## 9. 面试表达

> 做工业网络评估时，我会按拓扑资产、分区分域、通信矩阵、边界防护、远程运维、交换路由、工业网关、监测审计几个模块检查。重点不是单纯看设备是否在线，而是判断跨域路径是否合理、通信是否必要、边界是否最小开放、远程入口是否受控。
