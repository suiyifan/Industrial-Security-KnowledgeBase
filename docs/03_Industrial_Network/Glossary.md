# 03 章术语表

## Purdue Model

工业控制系统分层参考模型，用于描述企业 IT、生产管理、监控控制、基本控制和现场过程。

## Zone

安全区域，一组安全需求相似的资产。

## Conduit

区域之间的受控通信路径。

## Industrial DMZ

工业隔离区，位于办公网和控制网之间，用于数据中转、远程运维、补丁和文件交换。

## VLAN

虚拟局域网，用于在交换机上进行逻辑网络划分。

## ACL

访问控制列表，用于限制不同地址、端口或协议的通信。

## Firewall

防火墙，用于控制网络边界通信。

## Industrial Firewall

工业防火墙，可结合工业协议功能进行更细粒度控制。

## Communication Matrix

通信矩阵，记录资产之间的通信关系、协议、端口、方向和业务说明。

## Jump Server

跳板机，用于集中运维访问。

## Bastion Host

堡垒机，用于账号管理、访问控制和会话审计。

## VPN

虚拟专用网络，常用于远程接入。

## SPAN

交换机端口镜像，把某些流量复制到监测端口。

## TAP

网络分光设备，用于复制链路流量。

## IDS

入侵检测系统，用于发现异常行为和攻击迹象。

## Industrial IDS

工业入侵检测系统，关注工业协议、控制操作、工程下载、PLC 状态变化等。

## Protocol Gateway

协议网关，用于不同工业协议或网络之间转换。

## Serial Server

串口服务器，用于将串口设备接入以太网。

## Remote Maintenance

远程运维，厂商或维护人员通过远程通道进入生产环境进行维护。

## Attack Path

攻击路径，攻击者从入口到目标资产的连续步骤。
