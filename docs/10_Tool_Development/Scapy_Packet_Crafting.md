# Scapy 数据包构造

## 1. Scapy 的定位

Scapy 适合构造、解析和修改数据包。它在安全研究中常用于：

- 构造协议测试包。
- 从 PCAP 中提取字段。
- 验证协议字段变化。
- 编写最小触发样本。
- 做轻量级流量分析。

## 2. 工控协议中的用法

对工业协议，Scapy 可以帮助你：

- 构造 TCP/IP 层。
- 读取和修改 payload。
- 从 PCAP 中提取 Modbus、S7、私有协议字段。
- 快速生成边界测试样本。

如果协议已有成熟库，优先用成熟库；如果是私有协议或字段实验，Scapy 更灵活。

## 3. 构造包的安全边界

- 明确目标是仿真环境还是授权测试环境。
- 控制发包速率。
- 不默认发送破坏性 payload。
- 对写操作增加显式确认。
- 保存发送样本，便于复盘。

## 4. 输出设计

工具输出应包含：

- 发送时间。
- 目标地址。
- 协议字段。
- 原始十六进制。
- 响应摘要。
- 异常或超时原因。

## 5. 面试表达模板

> 我使用 Scapy 更多是为了构造最小协议样本和验证字段影响，而不是盲目发包。比如协议逆向时，我会从正常 PCAP 里提取 payload，修改长度、功能码或序列号，再观察响应变化。对真实工控设备要控制速率和动作范围，优先在仿真环境验证。
