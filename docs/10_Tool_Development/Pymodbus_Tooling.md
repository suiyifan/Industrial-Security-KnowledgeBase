# pymodbus 工具开发

## 1. pymodbus 能做什么

`pymodbus` 是 Python 中常用的 Modbus 协议库，可用于：

- 搭建 Modbus TCP 客户端。
- 搭建本地仿真服务。
- 读取线圈、离散输入、保持寄存器、输入寄存器。
- 练习功能码和异常响应。
- 做只读资产探测和协议验证。

## 2. Modbus 工具项目建议

### 2.1 只读寄存器读取器

参数：

- `--host`
- `--port`
- `--unit-id`
- `--address`
- `--count`
- `--timeout`

输出：

- 目标。
- 功能码。
- 地址范围。
- 返回值。
- 异常码。
- 耗时。

### 2.2 Modbus 资产探测器

只做低风险动作：

- TCP 连接测试。
- 读取设备标识。
- 读取少量测试寄存器。

避免默认写线圈、写寄存器。

## 3. 异常响应理解

Modbus 异常码很重要：

- `01`：非法功能。
- `02`：非法数据地址。
- `03`：非法数据值。
- `04`：从站设备故障。

工具输出应该把异常码翻译成人能读懂的含义，而不是只打印原始数字。

## 4. 工控安全价值

这类工具能帮助：

- 验证资产是否开放 Modbus。
- 检查哪些地址可读。
- 理解功能码与寄存器关系。
- 为后续流量分析和检测规则提供样本。

## 5. 面试表达模板

> 我会用 pymodbus 做只读型 Modbus 工具，比如读取设备标识或少量寄存器，并把功能码、地址、数量、异常码和耗时结构化输出。工控现场我不会默认执行写线圈、写寄存器这类动作，工具会控制速率并要求授权环境使用。
