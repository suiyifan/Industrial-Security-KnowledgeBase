# 10 Tool Development

岗位要求包括工具开发，建议围绕协议、安全测试和报告自动化做小工具。

优先项目：

1. TCP echo client/server
2. 简单端口扫描器
3. Modbus TCP 请求构造器
4. Modbus TCP 资产探测器
5. PCAP 摘要分析器
6. CVE 信息整理脚本
7. [仿真与工具路线：把理论变成可验证经验](./Simulation_and_Tooling_Roadmap.md)

代码原则：

- 参数清晰
- 输出可读
- 异常处理明确
- README 写清楚用途、输入、输出和限制
