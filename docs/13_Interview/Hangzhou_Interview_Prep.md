# 杭州面试准备

## 1. 地点表达

可以这样说：

> 我对杭州是有明确意向的。杭州在安全、互联网、智能制造和工业数字化方向都有比较多机会，也适合长期技术发展。这个岗位方向和我目前的学习投入匹配，所以地点不是短期尝试，而是我愿意认真考虑和投入的方向。

## 2. 岗位匹配表达

围绕 JD 说：

- 工控系统软件、协议、设备安全研究。
- 数据挖掘、软件逆向、协议逆向、漏洞挖掘、工具开发。
- 跟踪漏洞、攻击手段、防护产品。
- 安全测试、评估、审计。
- 安全分析报告。

回答模板：

> 我看这个岗位不是单纯 Web 安全，而是更偏工控安全研究，包含协议、设备、逆向、漏洞和工具开发。我目前的准备也是按这个方向拆的，所以比较匹配。

## 3. 初级候选人优势

不要假装资深，可以强调：

- 学习路径清晰。
- 愿意从基础任务做起。
- 有文档和作品集沉淀。
- 知道工控测试边界。
- 有持续投入意愿。

## 4. 可能被问

- 为什么从当前方向转工控安全？
- 是否接受杭州工作？
- 是否能接受出差或项目现场？
- 真实工控经验不足怎么办？
- 期望薪资如何？

## 5. 期望薪资表达

可以保守而开放：

> 我更看重岗位方向和成长空间，薪资希望结合我的能力、岗位要求和公司体系来定。如果面试后双方匹配，我也愿意根据整体 offer 情况沟通。
