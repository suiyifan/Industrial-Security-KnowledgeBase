# 13 Interview：面试准备

本章用于把知识库内容转化成面试表达：自我介绍、HR 问答、技术问答、项目经历、简历 bullet 和模拟面试清单。

## 本章能力目标

- 能用 1-2 分钟讲清楚自己为什么匹配工控安全研究员岗位。
- 能把项目经历讲成 “背景、方法、证据、结果、反思”。
- 能回答 Web、协议、逆向、漏洞、工控体系、工具开发相关问题。
- 能面对 HR 问杭州、转岗、学习路径、稳定性等问题。

## 表达原则

- 不夸大。
- 不说没做过的现场经历。
- 用仿真、公开样本、代码和报告证明能力。
- 每个回答尽量带一个具体项目或方法。
- 主动说明安全边界和验证意识。
