# 模拟面试检查清单

## 面试前

- 准备 30 秒、1 分钟、2 分钟自我介绍。
- 准备 2 个项目详细讲法。
- 准备 1 个失败或踩坑经历。
- 准备岗位匹配说明。
- 熟悉 JD 中每个关键词。
- 确认 GitHub 作品集链接可打开。

## 技术准备

- 能讲清 PLC、SCADA、HMI、DCS。
- 能讲清 Modbus TCP 报文结构。
- 能讲清常见 Web 漏洞。
- 能讲清 IDA/Ghidra 基本流程。
- 能讲清 CVE 分析闭环。
- 能讲清 IEC 62443 和分区分域。
- 能讲清工具开发项目。

## 项目表达

每个项目必须能回答：

- 为什么做。
- 怎么做。
- 输入是什么。
- 输出是什么。
- 遇到什么问题。
- 和岗位有什么关系。
- 有什么安全边界。
- 后续如何改进。

## HR 准备

- 为什么杭州。
- 为什么工控安全。
- 经验不足怎么办。
- 是否接受基础任务。
- 是否接受项目节奏。
- 期望薪资。

## 面试后复盘

- 记录被问问题。
- 标记答得不好的地方。
- 补充知识库。
- 更新项目 README。
- 准备下一轮更好的表达。
