# 术语表

| 术语 | 解释 |
| --- | --- |
| ICS | Industrial Control System，工业控制系统 |
| OT | Operational Technology，运行技术 |
| IEC 62443 | 工业自动化和控制系统安全标准体系 |
| Zone | 区域，安全需求相近的资产集合 |
| Conduit | 通道，不同区域之间的受控通信路径 |
| Purdue Model | 工业网络分层模型 |
| Industrial DMZ | 工业隔离区，用于办公网和生产网之间的缓冲 |
| 纵深防御 | 多层安全控制互相补充的防御思想 |
| 威胁建模 | 分析攻击者、入口、路径、资产和影响的方法 |
| ATT&CK for ICS | MITRE 针对工控攻击技术的知识库 |
| SIS | Safety Instrumented System，安全仪表系统 |
| 联锁 | 防止危险状态或错误操作的保护逻辑 |
| 强制输出 | 人为覆盖 PLC 输出状态的操作 |
| 白名单 | 只允许明确授权通信或行为的策略 |
| 被动识别 | 不主动探测，通过流量和日志识别资产 |
| 应急响应 | 对安全事件进行确认、隔离、取证、清除和恢复的过程 |
