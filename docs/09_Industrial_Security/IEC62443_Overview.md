# IEC 62443 与工控安全体系

## 核心思想

IEC 62443 是工业自动化与控制系统安全的重要标准体系。入门阶段先理解：

- 资产识别
- 区域与管道
- 分区分域
- 纵深防御
- 安全等级
- 风险评估
- 访问控制
- 日志与监测

## 与岗位的关系

岗位不仅要求发现漏洞，还要求安全测试、评估、审计和报告输出。IEC 62443 可以帮助把技术问题转化为体系化风险表达。

## 实践任务

选择一个模拟工业网络，输出：

- 资产清单
- 网络区域
- 关键通信路径
- 潜在威胁
- 安全控制措施
- 剩余风险
